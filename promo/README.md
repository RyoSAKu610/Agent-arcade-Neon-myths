# NEON MYTHOS: Lumen Atlas PV

This folder contains a 90-second promotional video for the Lumen Atlas adventure.

- `neon-mythos-ledger-dawn-pv.mp4`: lightweight 720p clean PV without subtitles
- `neon-mythos-ledger-dawn-pv-poster.jpg`: video poster frame
- `generate_pv.py`: reproducible renderer using the repository sprites and a credited CC BY music source
- `credits.md`: music attribution and license details
